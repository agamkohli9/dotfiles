# Dotfiles
These are my dotfiles for i3, picom, and polybar on my Arch Linux PC

Check out https://www.reddit.com/r/unixporn/comments/oaaw1g/i3gaps_i_like_transparent_polybar/

To use on your own Arch Linux PC, install:

* picom-tryone-git (transparency effect for terminal emulator and polybar)
* i3-gaps (window manager)
* st (terminal emulator)
* polybar (bar on top)
* fish (shell)
